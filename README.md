# 快速搭建支持markdown文件的文档


## 安装
```
yarn || npm install 
```

### Compiles and hot-reloads for development
```
yarn || npm run serve
```

### Compiles and minifies for production
```
yarn || npm  run build
```

### Run your tests
```
yarn || npm run test
```

### Lints and fixes files
```
yarn || npm  run lint
```
